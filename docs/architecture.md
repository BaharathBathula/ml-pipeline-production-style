# Architecture

This repository demonstrates a modular production-style machine learning workflow.

## Components

1. Data Ingestion
2. Data Validation
3. Data Transformation
4. Model Training
5. Model Evaluation
6. Model Packaging
7. FastAPI Model Serving
8. CI/CD Automation

## Flow

Raw CSV -> Train/Test Split -> Validation -> Preprocessing -> Training -> Metrics -> Model Artifact -> API Serving
